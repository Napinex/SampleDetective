# Bestellprozess und Second-Hand-Sample-Market

Der Bestellprozess beginnt mit einer Musteranforderung: Materialnummer, gewünschter Test, Zieltermin,
Priorität und Projektkontext. Bevor eine Neubestellung ausgelöst wird, wird geprüft, ob ein vorhandenes
Prüfmuster wiederverwendet werden kann.

Prüfschritte:
1. Exakte Materialnummer suchen
2. BOM-ähnliche Muster suchen
3. Verfügbarkeit und verfügbare Zeitfenster prüfen
4. Defects prüfen
5. Health Score und Usage Hours prüfen
6. Erlaubte Tests prüfen
7. Reuse oder Neubestellung empfehlen

Wenn ein vorhandenes Muster passt, wird Reuse empfohlen. Wenn nicht, wird Neubestellung empfohlen.
