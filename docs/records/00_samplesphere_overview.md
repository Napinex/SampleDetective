# SampleSphere AI – Second-Hand Sample Market Assistant

SampleSphere AI ist ein lokaler RAG-/LLM-Assistent für das Prüfmuster-Lebenszyklusmanagement.
Das System verbindet Prüfmuster, BOM-Konfiguration, Verfügbarkeit, Tests, Defects, Usage Hours,
Health Score, Reuse Score und Bestellentscheidung.

Ziel:
- vorhandene Prüfmuster schneller finden
- ähnliche Konfigurationen anhand der BOM vergleichen
- Wiederverwendung gegenüber Neubestellung bewerten
- bereits gelaufene Tests und noch erlaubte Tests transparent machen
- Fragen in natürlicher Sprache beantworten

Der Datenbestand ist für den Assistenzprozess konsolidiert und auf Prüfmuster, Tests, Defects und Wiederverwendungsentscheidungen ausgerichtet.
